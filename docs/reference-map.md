# Reference Map

The following repos were cloned into `vendor/reference/` as source material for patterns and not as runtime dependencies.

## Reused patterns

- `mercari-watchdog`: poll a marketplace page, diff new listings, notify fast.
- `pkmn-tcg-app`: price alerts and local data storage.
- `Trading-Card-Scanner`: detect card, identify it, then look up pricing.
- `fetch-psa-api`: PSA cert lookup and image retrieval.
- `justtcg-js`: typed pricing API client and response normalization.

## Reference-only repos

- `vault`
- `psa-scrape`
- `gopsa`
- `Pokemon-Card-Scanner`
- `cardex`
- `card-scanner`

These are kept for architectural reference only. They are not imported into the build.
